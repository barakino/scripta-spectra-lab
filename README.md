# Scripta/Spectra Lab

Website for the **Scripta/Spectra Lab** — Computational Analysis and Imaging for
Cultural Heritage — directed by Barak Sober, The Hebrew University of Jerusalem.

Live site: https://barakino.github.io/scripta-spectra-lab/

Plain static HTML/CSS/JS — no build step. The source, generator, and CV-sync
pipeline live in a separate working repo; this repository holds the published
site only.
